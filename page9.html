Malware By Philip Wu

Malware, or malicious software are unwanted software that is installed in your system without your consent. It can attach itself to legitimate code, or lurk in useful applications and replicate itself across the Internet.)

File Infectors
These viruses attach themself the executable code, such as the most widely known .exe files. The virus is installed when the code is loaded, or when the .exe file is run. Another version of a file injector associates itself with a file by creating a virus file with the same name, but an .exe extension. 


Macro viruses
These viruses infect applications such as Microsoft Word or Excel by attaching themselves to an initialization sequence. When the application is opened, the virus executes instructions before transferring control to the application. The virus then replicates itself, and attaches to other code in the system.


System/Boot-record infectors
A boot-record virus attaches itself to the master boot record on hard disks. When the system is started, it looks at the boot sector and loads the virus into it's memory, where it will spread to other disks and computers.


Polymorphic viruses
These viruses conceal themselves through varying cycles of encryption and decryption. The encrypted virus and associated mutation engine are initially decrypted by decryption programs, causing the virus to activate and infect an area of the code. The mutation engine then develops a new decryption routine, and the virus encrypts the mutation engine and a copy of the virus with an algorithm corresponding to the aforementioned routine. The encrypted mutation engine and virus is attached to new code, and the process repeats.


Stealth viruses
These viruses take over system functions to conceal themselves. This is done through compromising malware detection software so that the software falsely reports an infected area as uninfected. These viruses also conceal any increase in the size of an infected file, or changes to the file's date and last time of modification.


Trojans
Named after the Trojan horse, this malware hides themselves in useful programs. This establishes a back door that can be exploited by attackers, such as opening a high-numbered port so the hacker can use it to listen and perform attacks.
 - One major difference from this and viruses is that Trojans do not self-replicate.


Logic bombs
A logic bomb is a malicious software that is appended to an application, and is only triggered by specific occurrences, such as a logical condition or a specific date and time.


Worms
Worms differ from viruses in the way that they are not attached to a host file, but are self-contained programs that spread across networks and computers 

Malware → Virus
(Viruses are a type of malware that are created to alter the way a computer operates, and usually designed to be able to spread itself to other computers. A virus operates by inserting or attaching itself to legitimate programs that support external scripts to execute its code without seeming suspicious. During this process, a virus has the potential to cause damaging effects, such as corrupting or destroying data.)

ILOVEYOU virus
A virus using emails as a means of transmission, it disguised itself as a love letter from a contact of the victim. The malware itself was hidden in an attachment named "LOVE-LETTER-FOR-YOU.TXT.vbs". Clicking on it would activate a script, which would cause the worm to start overwriting random files on the user's PC. The worm would also proceed to send copies of itself to all contacts in the user's address book.
ILOVEYOU infected over 10 million Windows PCs. It caused the Pentagon, the British Gov't, and the CIA to completely shut down their mail systems.

Takeaways: 
- Be careful when opening attachments that are mailed to you, or in general, all files that are downloaded off the Internet.
- Don't click on programs you aren't sure are secure.
- Say NO to any software that comes attached with free software online. Most of the time, it's malicious, or will advertise itself maliciously.
